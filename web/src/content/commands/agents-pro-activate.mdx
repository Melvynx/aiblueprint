---
title: agents pro activate
description: Activate AIBlueprint CLI Premium with an access token.
---

Use `agents pro activate` to store a premium access token.

```bash
aiblueprint agents pro activate <token>
```

## Token argument

The token is optional in the command signature. If omitted, the command can prompt for it.

```bash
aiblueprint agents pro activate
```

## After activation

Use [agents pro status](/commands/agents-pro-status) to confirm the token state before installing premium configuration.
