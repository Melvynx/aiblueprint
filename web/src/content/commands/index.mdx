The CLI is organized around command families. Use `agents` for setup and rendering, `configs` for snapshots, and `openclaw` for OpenClaw configuration.

```bash
aiblueprint <family> <command>
```

## Shared folder options

The `agents`, `ai-coding`, `claude-code`, and `configs` command families can target custom folders.

```bash
aiblueprint agents setup --folder ~/workspace
aiblueprint configs list --agentsFolder ~/.agents
```

See [Folder options](/reference/folder-options) for the full list.
