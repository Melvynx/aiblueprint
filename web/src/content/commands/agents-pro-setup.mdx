---
title: agents pro setup
description: Install premium configurations after activation.
---

Use `agents pro setup` to install premium AIBlueprint configuration.

```bash
aiblueprint agents pro setup
```

## Requirements

Premium setup requires activation first.

```bash
aiblueprint agents pro activate
aiblueprint agents pro setup
```

## Folder behavior

The command uses parent folder options such as `--folder`, `--claudeCodeFolder`, `--codexFolder`, and `--agentsFolder`.
