---
title: agents pro
description: Manage AIBlueprint CLI Premium features.
---

`agents pro` groups premium AIBlueprint commands.

```bash
aiblueprint agents pro <command>
```

## Subcommands

- `activate [token]`
- `status`
- `setup`
- `update`
- `sync`

## Folder behavior

Premium setup, update, and sync commands use the same parent folder options as `agents`.
