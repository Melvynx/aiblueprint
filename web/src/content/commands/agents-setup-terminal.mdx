Use `agents setup-terminal` to configure the shell environment used alongside AI coding tools.

```bash
aiblueprint agents setup-terminal
```

## What it does

- Sets up terminal dependencies and shell shortcuts.
- Installs Oh My ZSH related configuration.
- Applies a themed terminal setup.

## Common usage

```bash
aiblueprint agents setup-terminal
aiblueprint agents setup-terminal --skip
```

## Folder behavior

The command reads the parent `--folder` option as the home directory for terminal setup.
