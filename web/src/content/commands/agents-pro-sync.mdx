---
title: agents pro sync
description: Sync premium configurations with selective update.
---

Use `agents pro sync` for selective premium configuration updates.

```bash
aiblueprint agents pro sync
```

## Difference from update

`update` refreshes premium configuration broadly. `sync` is intended for selective synchronization where existing local state matters.

## Common usage

```bash
aiblueprint agents pro sync
aiblueprint agents pro sync --agentsFolder ~/.agents
```
