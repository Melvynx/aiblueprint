---
title: agents backup load
description: Load a previous AI coding configuration backup interactively.
---

Use `agents backup load` to restore a previous backup through an interactive flow.

```bash
aiblueprint agents backup load
```

## What it restores

The command targets the selected Claude Code and shared agents folders.

## Common usage

```bash
aiblueprint agents backup load
aiblueprint agents backup load --folder ~/Developer/workbench
```
