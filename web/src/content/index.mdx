`aiblueprint` is a command-line tool for setting up AI coding environments. It installs opinionated defaults, links compatible configuration folders, renders shared agent definitions, and keeps named configs recoverable.

```bash
npm install -g aiblueprint-cli
aiblueprint agents setup
```

## Command families

- `aiblueprint agents` manages AI coding configuration setup.
- `aiblueprint ai-coding` is a legacy alias for `agents`.
- `aiblueprint claude-code` is a legacy alias for `agents`.
- `aiblueprint configs` saves, restores, lists, and backs up local configuration folders.
- `aiblueprint openclaw` manages OpenClaw-specific configuration.

## Default folders

Most commands operate from your home folder by default and target:

- `.claude/` for Claude Code configuration.
- `.codex/` for Codex configuration.
- `.agents/` for shared skills and agents.

Use the folder options when your configuration lives somewhere else.

```bash
aiblueprint agents setup --folder ~/Developer/workbench
aiblueprint configs save daily --codexFolder ~/custom-codex
```
