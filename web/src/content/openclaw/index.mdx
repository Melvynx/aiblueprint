---
title: openclaw
description: OpenClaw configuration commands.
---

`openclaw` groups OpenClaw-specific configuration commands.

```bash
aiblueprint openclaw <command>
```

## Options

| Option | Description |
| --- | --- |
| `-f, --folder <path>` | Custom OpenClaw folder path. Defaults to `~/.openclaw`. |

## Subcommands

- `pro activate [token]`
- `pro status`
- `pro setup`
- `pro update`
