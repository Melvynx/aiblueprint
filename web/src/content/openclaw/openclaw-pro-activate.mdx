---
title: openclaw pro activate
description: Activate OpenClaw Pro with an access token.
---

Use `openclaw pro activate` to store an OpenClaw Pro access token.

```bash
aiblueprint openclaw pro activate <token>
```

## Token argument

The token is optional in the command signature. If omitted, the command can prompt for it.

```bash
aiblueprint openclaw pro activate
```
