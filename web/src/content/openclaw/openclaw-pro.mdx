---
title: openclaw pro
description: Manage OpenClaw Pro features.
---

`openclaw pro` groups OpenClaw Pro commands.

```bash
aiblueprint openclaw pro <command>
```

## Subcommands

- `activate [token]`
- `status`
- `setup`
- `update`

## Folder behavior

The parent `openclaw --folder` option controls the OpenClaw folder path.
