The CLI registers two legacy aliases for `agents`.

| Alias | Equivalent |
| --- | --- |
| `ai-coding` | `agents` |
| `claude-code` | `agents` |

## Examples

```bash
aiblueprint ai-coding setup
aiblueprint claude-code codex-agents
```

These aliases expose the same setup, terminal, symlink, unify, Codex agent, premium, backup, and dynamic script commands.
