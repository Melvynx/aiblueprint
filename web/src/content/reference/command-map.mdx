---
title: command map
description: Compact reference for every static command registered by the CLI.
---

| Command | Purpose |
| --- | --- |
| `agents setup` | Setup AI coding configuration with AIBlueprint defaults. |
| `agents setup-terminal` | Setup terminal with Oh My ZSH, plugins, and a theme. |
| `agents symlink` | Create symlinks between AI coding tools. |
| `agents unify` | Unify skills and agents into `.agents`, then render Codex TOML agents. |
| `agents codex-agents` | Render shared Markdown agents into Codex TOML custom agents. |
| `agents pro activate [token]` | Activate AIBlueprint CLI Premium. |
| `agents pro status` | Check Premium token status. |
| `agents pro setup` | Install premium configurations. |
| `agents pro update` | Update premium configurations. |
| `agents pro sync` | Sync premium configurations with selective update. |
| `agents backup load` | Load a previous backup interactively. |
| `configs save <name>` | Save current config folders as a named config. |
| `configs load <name>` | Load a named config and backup current folders first. |
| `configs undo` | Undo the most recent config load. |
| `configs list` | List saved named configs. |
| `configs backups list` | List automatic backups with reasons. |
| `configs backups load <name>` | Load an automatic backup. |
| `configs backups create [reason]` | Create a manual backup. |
| `openclaw pro activate [token]` | Activate OpenClaw Pro. |
| `openclaw pro status` | Check OpenClaw Pro token status. |
| `openclaw pro setup` | Install OpenClaw Pro configurations. |
| `openclaw pro update` | Update OpenClaw Pro configurations. |
| Dynamic scripts | Register local Claude script commands at runtime. |
