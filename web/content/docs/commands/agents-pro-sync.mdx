---
title: agents pro sync
description: Analyze and selectively sync premium configuration changes.
command: aiblueprint agents pro sync
---

`agents pro sync` compares local configuration against premium assets and lets you choose what to apply.

```bash
aiblueprint agents pro sync
```

## What it reports

- New files.
- Modified files.
- Deleted files.
- Migration candidates.
- Unchanged files.

Use this instead of a broad update when you want control over individual premium changes.
