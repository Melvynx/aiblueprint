---
title: agents pro activate
description: Activate AIBlueprint CLI Premium with an access token.
command: aiblueprint agents pro activate
---

`agents pro activate` validates and saves a premium access token.

```bash
aiblueprint agents pro activate [token]
```

## Usage

Pass the token directly or omit it to enter the token interactively.

```bash
aiblueprint agents pro activate pro_token_here
aiblueprint agents pro activate
```

After activation, run `agents pro setup` or `agents pro sync`.
