`openclaw` manages OpenClaw configuration.

```bash
aiblueprint openclaw [options] [command]
```

## Options

| Option | Description |
| --- | --- |
| `-f, --folder <path>` | Custom OpenClaw folder path. Defaults to `~/.openclaw`. |

## Subcommands

| Command | Description |
| --- | --- |
| `pro` | Manage OpenClaw Pro features. |
