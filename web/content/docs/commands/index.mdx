The CLI is organized around command families. `agents` is the primary family, `configs` handles snapshots, and `openclaw` handles OpenClaw-specific premium setup.

## Top-level commands

| Command | Description |
| --- | --- |
| `aiblueprint agents` | AI coding configuration commands. |
| `aiblueprint ai-coding` | Legacy alias for `agents`. |
| `aiblueprint claude-code` | Legacy alias for `agents`. |
| `aiblueprint configs` | Save, load, undo, and inspect configuration snapshots. |
| `aiblueprint openclaw` | OpenClaw configuration commands. |

## Help

Every command supports `--help`.

```bash
aiblueprint --help
aiblueprint agents --help
aiblueprint configs backups --help
```
