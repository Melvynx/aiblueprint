---
title: openclaw pro setup
description: Install OpenClaw Pro configurations into the OpenClaw folder.
command: aiblueprint openclaw pro setup
---

`openclaw pro setup` installs OpenClaw Pro assets.

```bash
aiblueprint openclaw pro setup
```

## Target folder

The default target is `~/.openclaw`. Override it from the parent command.

```bash
aiblueprint openclaw --folder /tmp/openclaw pro setup
```
