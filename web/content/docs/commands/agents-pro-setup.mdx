---
title: agents pro setup
description: Install premium AIBlueprint configurations after activation.
command: aiblueprint agents pro setup
---

`agents pro setup` installs premium configuration files.

```bash
aiblueprint agents pro setup
```

## Requirements

Premium setup requires a valid token saved by `agents pro activate`.

## Folder overrides

```bash
aiblueprint agents --folder /Users/me pro setup
```
