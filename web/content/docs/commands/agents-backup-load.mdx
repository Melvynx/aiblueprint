---
title: agents backup load
description: Interactively load a previous automatic AIBlueprint backup.
command: aiblueprint agents backup load
---

`agents backup load` restores an automatic backup.

```bash
aiblueprint agents backup load
```

## Flow

1. Scan for available backups.
2. Select a backup.
3. Confirm the restore.
4. Back up the current configuration.
5. Restore the selected backup.
