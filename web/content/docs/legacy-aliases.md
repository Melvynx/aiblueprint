# Legacy aliases

## `aiblueprint ai-coding [options] [command]`

Legacy alias for `aiblueprint agents`. It supports the same options and nested commands.

Examples:

```bash
aiblueprint ai-coding setup
aiblueprint ai-coding --help
```

## `aiblueprint claude-code [options] [command]`

Legacy alias for `aiblueprint agents`, kept for older Claude Code setup flows.

Examples:

```bash
aiblueprint claude-code setup
aiblueprint claude-code pro status
```
